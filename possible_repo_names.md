# Possible Repo Names

## Original idea
- **POSdown** — POS + markdown

## Roll-based
- **Rolldown** — paper roll + markdown *(conflict: Rust bundler library)*
- **Rollout** — roll of paper + printing output
- **Unroll** — what you do with a receipt roll
- **Reroll** — fun, slightly cheeky
- **Rollmark** — roll + markdown mark
- **Rollprint** — straightforward
- **Rollio** — playful
- **Rolltape** — ticker tape feel

## Down-based
- **Pressdown** — printing press + markdown
- **Feeddown** — paper feed + markdown
- **Stripdown** — receipt strip + markdown
- **Scrolldown** — paper scroll + markdown, sidesteps Rolldown conflict

## Tape / ticker
- **Tapemark** — ticker tape + markdown
- **Tapescript** — ticker tape + script/receipt
- **Tapesque** — ticker tape + -esque suffix

## -esque words
- **Burnesque** — thermal printing burns the paper; near-rhyme with *burlesque* ⭐
- **Scrollesque** — paper scroll, flows naturally
- **Reelesque** — paper reel
- **Chitesque** — chit is an old word for a receipt/slip, obscure but charming
- **Slipesque** — receipt slip
- **ESCesque** — ESC printer command baked into the suffix; reads like "Kafkaesque" for printer nerds

## Miscellaneous
- **Slipdown** — receipt slip + markdown
- **Slipmark** — receipt slip + markdown mark
- **Burnmark** — thermal burn + markdown mark
- **Heatset** — thermal heat + typesetting
