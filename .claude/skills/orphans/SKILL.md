---
name: orphans
description: "Generates a list of orphaned components or files. Usage: /orphans"
model: sonnet
---

When running this skill, scan through the key Nuxt folders to identify javascript, typescript, sass or vue files that appear to be orphaned.

Add them to the file `orphans.md`

In the file, create two bulleted lists:

1) Javascript Files
2) Vue Components
