# Coding Style for SASS Files

## General

Indented SASS is always used

All classes and ids should be all lowercase

Word separators for classes and ids should be underscores


# Path at the top

The relative path in the folder structure should be placed in a comment in the first line of the file

For example:

```sass
// /_sass/global.sass
```
